== Description ==

The plugin facilitates the user can add the progress counter block.
It allows to change counter value, text color, incomplete progress color and complete progress color.
Using the block you can easily move to bar counter to circle counter.

= Key Features: =

* Easy to use
* Works seamlessly with other WordPress Plugins
* Simple to change color settings


== Installation ==

* Download the plugin
* Upload the folder "nk-progress-counter-block" to wp-content/plugins (or upload a zip through the Wordpress admin)
* Activate and enjoy!

== Frequently Asked Questions ==

= In which WordPress version this Plugin is compatible?

It is compatible from 5.0 WordPress version.

= Can I change the progress color?

Yes, you can change the progress incomplete color, progress complete color and text color.
